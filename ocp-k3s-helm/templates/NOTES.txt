📈 Monitor Deployment:

   Watch all Jobs:
   $ oc get jobs -n {{ .Release.Namespace }} -l app.kubernetes.io/instance={{ .Release.Name }} -w
   
   Watch VirtualMachines:
   $ oc get vm -n {{ .Release.Namespace }} -w
   
   Watch VirtualMachineInstances:
   $ oc get vmi -n {{ .Release.Namespace }} -w
   
   View everything together:
   $ watch 'oc get job,vm,vmi -n {{ .Release.Namespace }}'

🌐 K3s API Access:

   After deployment completes, get the external URL:
   $ oc get route {{ .Release.Name }}-k3s-api -n {{ .Release.Namespace }} -o jsonpath='{.spec.host}' && echo
   
   Or view full route details:
   $ oc get route {{ .Release.Name }}-k3s-api -n {{ .Release.Namespace }}

📥 Download Kubeconfig:

   1. Wait for Job-4 to complete (may take 3-5 minutes after install):
      $ oc wait --for=condition=complete job/{{ .Release.Name }}-04-fetch-kubeconfig \
          -n {{ .Release.Namespace }} --timeout=600s
   
   2. Download the kubeconfig file:
      $ oc get secret {{ .Release.Name }}-kubeconfig \
          -n {{ .Release.Namespace }} \
          -o jsonpath='{.data.kubeconfig}' | base64 -d > {{ .Values.clusterName }}-kubeconfig.yaml
   
   3. Use the kubeconfig:
      $ export KUBECONFIG={{ .Values.clusterName }}-kubeconfig.yaml
      $ kubectl get nodes
      $ kubectl get pods -A

🔌 Access Virtual Machines:

   Connect to Master-0 console (requires virtctl):

   SSH to Master-0 (requires virtctl):
   $ virtctl ssh {{ .Values.username }}@{{ .Release.Name }}-master-0 -n {{ .Release.Namespace }}

🆘 Troubleshooting:

   If a Job fails:
   1. Check job logs (commands above)
   2. Check VM/VMI status: oc describe vm <vm-name>
   3. Check PVC status: oc get pvc -n {{ .Release.Namespace }}
   
   If Master-0 fails to start:
   $ oc get events -n {{ .Release.Namespace }} --sort-by='.lastTimestamp'
   
   To restart a failed job:
   $ oc delete job {{ .Release.Name }}-XX-<job-name> -n {{ .Release.Namespace }}
   $ helm upgrade {{ .Release.Name }} k3s-openshift-virt -n {{ .Release.Namespace }} --reuse-values

🗑️  Uninstall:

   Remove the entire K3s cluster:
   $ helm uninstall {{ .Release.Name }} -n {{ .Release.Namespace }}
   
   This will delete:
   - All Jobs
   - All VirtualMachines
   - Service and Route
   - Secrets (credentials, token, kubeconfig)
   
   Note: PVCs are retained by default. To delete them:
   $ oc delete pvc -l app.kubernetes.io/instance={{ .Release.Name }} -n {{ .Release.Namespace }}
